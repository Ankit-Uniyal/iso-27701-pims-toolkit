# 10 — Annex A: PII Controller Controls

This folder contains implementation guidance and control templates for organisations acting as **PII Controllers** under ISO/IEC 27701:2019 Annex A.

## What is Annex A?

Annex A provides additional guidance and controls for organisations that determine the **purposes and means** of PII processing (PII Controllers). These controls extend ISO/IEC 27001 with specific privacy requirements.
