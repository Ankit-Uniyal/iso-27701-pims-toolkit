## 1. Purpose

This Privacy Risk Register records all identified privacy risks arising from the organisation's PII processing activities. It is maintained in accordance with ISO/IEC 27701:2025 Clause 6.1 and the Privacy Risk Assessment Methodology (PIMS-PLN-003).

**Risk scoring uses a 5x5 Likelihood x Impact matrix. Full methodology is in PIMS-PLN-003.**

---

## 2. Risk Rating Summary

| Score Range | Rating | Action Required |
|---|---|---|
| 20-25 | CRITICAL | Halt/suspend processing; immediate DPO escalation; mandatory treatment |
| 12-19 | HIGH | Documented treatment plan within 30 days; DPO informed |
| 6-11 | MEDIUM | Treatment plan required; review at next management meeting |
| 2-5 | LOW | Accept with monitoring; review annually |
| 1 | MINIMAL | Accept; note; no further action |

## 4. Threat Category Reference

Use the following standardised threat categories when completing the Risk Description column:

| Threat Category | Examples |
|---|---|
| Unauthorised access | Data breaches, insider threats, credential compromise, account takeover |
| Unlawful processing | No valid legal basis, purpose limitation violations, consent invalid |
| Inaccuracy / data quality | Incorrect data leading to wrong decisions about individuals |
| Excessive retention | Data held beyond permitted retention periods |
| Cross-border violations | Transfers without adequacy decision or adequate safeguards; no TIA |
| Third-party failures | Processor non-compliance, sub-processor data misuse, contractor incident |
| Individual rights failures | Failure to respond to DSARs, erasure requests, objections within deadline |
| Consent failures | Invalid consent mechanisms, withdrawal not honoured, no consent refresh |
| Automated decision-making | Profiling without transparency, no right to contest, algorithm bias |
| Joint controller failure | Undefined responsibilities, missing JCA, liability gaps |
| Physical threats | Lost/stolen devices, paper records mishandling, unsecured disposal |
| Security incident | Ransomware, malware, system compromise affecting PII |
