## 3. Privacy-Specific Context Factors (ISO 27701:2025 Clause 4.1 Extension)

As required by ISO/IEC 27701:2025, the following privacy-specific context factors have been identified:

| Factor | Description | Reference |
|--------|-------------|-----------|
| Role(s) as PII Controller | Organisation determines purposes and means of PII processing | Annex A (A.7–A.8) |
| Role(s) as PII Processor | Organisation processes PII on behalf of third-party controllers | Annex B (B.8) |
| Joint Controller arrangements | Identified joint controller relationships requiring formal agreements | A.7.2.7 |
| PII categories processed | Identified in PII Processing Inventory (PIMS-CTX-005) | A.7.2.1 |
| Legal/regulatory environment | GDPR, UAE PDPL, UK GDPR, India DPDPA, CCPA — see Legal Register | PIMS-CTX-004 |
| Data subject geography | EU/EEA, UAE, UK, India, [other jurisdictions] | Cross-border procedure |
